Added `namespaces` to `attr.make_class` to provide additional attributes
for newly created class

```python
import attr

@attr.define
class A:
  pass

def new_echo(cls, *something: int) -> tuple[int, ...]:
  return something

B = attr.make_class("B", bases=(A,), namespaces={"echo": classmethod(new_echo)})

B.echo(1, 2, 3)
```
